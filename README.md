#  FoodShare

**FoodShare** is a community food sharing web application built with PHP, MySQL, HTML, CSS, and JavaScript.

##  Objective
To reduce food waste by connecting donors, receivers, volunteers, and admins through a single web platform.

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP (Procedural)
- Database: MySQL
- Server: Apache (via XAMPP)
- IDE: Visual Studio Code

##  Features
- Donor adds food donations and can modify according to its expiry
- Receiver requests available food with quantity selection and can track requests made and its states (Accepted,Assigned,Delivered)
- Admin approves or declines requests and assigns volunteers and auto logs activity in table
- Volunteer manages assigned deliveries
- Session-based login and role management
- Modern, responsive UI

##  Database
Import `community_food_sharing_login.sql` into phpMyAdmin to create required tables.
